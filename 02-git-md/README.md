# Računalniški praktikum

## Bližnjice na tipkovnici

Kopiraj označeno v odložišče: **Ctrl+C** (*C*opy)  
Izreži označeno v odložišče: **Ctrl+X**  
Prilepi vsebino odložišča: **Ctrl+V**

Dodatne bližnjice, ki sem se jih naučila:

- **Ctrl+Z** – razveljavi zadnje dejanje (*Undo*).  
  Uporabna je, kadar pomotoma izbrišem besedilo ali datoteko.

- **Ctrl+Shift+T** – znova odpri zadnji zaprti zavihek v brskalniku.  
  Pogosto pride prav, ko po nesreči zaprem pomembno stran.

- **Alt+Tab** – hitro preklopi med odprtimi programi ali okni.  
  Zdaj jo uporabljam za hitrejše delo, ko imam odprtih več aplikacij.
